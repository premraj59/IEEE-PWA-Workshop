# IEEE PWA WorkShop

### Pre-requisites

- Browser - Chrome / Firefox
- Nodejs > 8

```
$ curl -sL https://deb.nodesource.com/setup_10.x | sudo -E bash -
$ sudo apt-get install -y nodejs
```

- VSCode
- Postman
- SQLite

```
$ sudo apt-get install zlib1g:i386
$ #or
$ sudo apt-get install lib32z1
```
